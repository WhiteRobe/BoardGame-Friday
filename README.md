# 桌游-星期五

#### 项目介绍
《桌游-星期五》是一个基于Vue及其插件、Bootstrap、jQuery的单页面应用项目。

摸鱼的时候和朋友一起写出来的，由于是三个人写，加上对游戏规则的认知不同，反正没有用比较科学的方式写项目。
不过能玩就行。
